## Setup Instructions

### Setup Virtual Environment


```bash
pip install virtualenv
py -m venv env
.\env\Scripts\activate
```      

### Install Required Python Modules

```bash
pip install -r requirements.txt
```
### Start Web Server

```bash
python manage.py runserver
```
